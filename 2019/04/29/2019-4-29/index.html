今天做了哪些事情：

  • 复习了一下面向对象

    • name decoration

    • 默认值

      像函数这样里面的参数，如果不进行赋值，而且没有默认值的化，就是出现调用出错的提示，但是如果在定义函数的时候自己加上默认值，就不会出现这种情况

    • 宏定义macro 与 const

      1. 宏定义只是一个替换式，他只是一个符号
      2. const是有自己的变量类型的，只是不能改变,const 修饰的左边的值

    • static 静态变量

      在类里面定义一个静态变量，这个静态变量是被全局所共享的，也就是说，也就是可以用来对对象数进行计数，然后要注意的一点就是，static的定义需要在类的外面

    • inline 函数

      内联函数相比宏定义的优点：

      1. 宏调用并不执行类型检查，甚至连正常参数也不检查，但是函数调用却要检查
      2. C语言的宏使用的是文本替换，可能导致无法预料的后果，因为需要重新计算参数和操作顺序
      3. 类中的函数几乎都是内联的，所以不会写内联的关键字
